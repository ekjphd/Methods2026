Group Activity: Two Simultaneous Strands

Visual Analysis Strand (Rose)

Groups apply compositional, social, and technological analysis to a set of 8–10 images. [Dr. Johnson selects before class — from a social media account, news source, or DH project.]

  • What does each individual image argue?
  • What does the set argue collectively that no single image does?
  • What would be lost by describing these images in text?

Projects-as-Data Strand

Groups treat a DH project as the object of analysis. Suggested projects:

  • Viral Texts — reprinting and circulation of 19th-century news
  • Photogrammar — New Deal-era photography
  • Colored Conventions Project — 19th-century Black political organizing

Questions:

  • What claims does the project's design make? What argument does the interface embody?
  • What did the choice to visualize data this way foreground or suppress?
  • How does Underwood's critique of distant reading apply to this project's methodology?

Debrief

Where do the visual analysis and projects-as-data methods overlap? Where do they require completely different skills? What would it look like to combine them in a dissertation chapter?
